__GIT Command Line 명령어 정리__
==============================

### __1. 환경설정__
> - git config --global user.name "사용자명"
> 사용자명을 등록합니다 (필수)
> 
> - git config --global user.email "이메일주소" 
> 이메일 주소를 등록합니다. (필수)

### __2. 기본명령어__
> -  git init
> 현재 디렉토리에 git 저장소를 생성합니다.
> 
> - git status
> 파일의 상태를 확인합니다.
>
> - git add 파일명
> Untracked files을 Staging Area에 올립니다.
> - git commit -m "커밋 메시지"
> Staging Area에 올라가 있는 파일들을 Git Repository에 올립니다.
